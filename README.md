# 高質感 Zen 數獨 (The Zen Sudoku)

這是一個基於 Web 的高質感數獨遊戲，專注於極簡設計、流暢體驗與自我成長的成就系統。

## ✨ 特色功能

- **優雅視覺**：支援深色 (Dark) / 淺色 (Light) 模式，採用玻璃擬態 (Glassmorphism) 設計與舒適的森林色系。
- **成長系統 (XP & Levels)**：
  - **經驗值 (XP)**：根據難度、解題速度與連勝紀錄計算 XP。
  - **等級稱號**：從「🌱 見習者」到「✨ 數獨傳說」共 10 個等級。
  - **連勝紀錄**：追蹤連續勝利局數，連勝越高 XP 加成越多。
- **個人檔案與數據**：
  - 詳細統計：總局數、勝率、各難度最佳時間。
  - **資料備份 (Export/Import)**：支援將遊戲進度匯出為 `.json` 檔案，隨時在不同裝置恢復紀錄。
- **專業數獨引擎**：
  - 多種難度：從初級到數獨大師。
  - 邏輯校驗：支援唯一解驗證 (Hard 模式)。
  - 輔助功能：筆記模式、衝突顯示、剩餘數字計數。

## 🚀 如何啟動

本遊戲為純靜態網站，無需安裝任何依賴。

1. 下載所有檔案。
2. 直接點擊 `index.html` 即可在瀏覽器開始遊戲。
